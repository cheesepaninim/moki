# MOKI
작성한 게시글을 모듈로 다른 게시글에서 불러와 쓸 수 있는 블로그 

## NPM task
```shell script
## setup
$ npm install

## Runs the app in development mode.
$ npm start

## Builds the app for production to the build folder.
$ npm run build
```
